# YATA - Nest.js Backend

## Development

### Reguired software

* Node.js - recommended v18.13.0
* npm (recommended v8.19.3) or another package manager
* Nest.js CLI - recommended v9.2.0

### Installation

```bash
# installation
npm install
```

### Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

### Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
