# YATA - Angular Frontend

## Development

### Required software

- Node.js - recommended v18.13.0
- npm (recommended v8.19.3) or another package manager

Optional & recommended:

- Angular CLI

### Installation

```bash
npm install
```

### Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run watch
```

### Test

```bash
# unit tests
$ npm run test
```

### Linting and formatting code

```bash
$ npm run lint && npm run format
```
